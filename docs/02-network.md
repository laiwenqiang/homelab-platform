# 网络规划

> 目标：做到“隔离 + 可控访问 + 可排障”。

## 网段规划（占位，按你实际填写）
- mgmt: 192.168.10.0/24（PVE 管理、仅管理终端可达）
- tools: 192.168.20.0/24（日常工具 LXC/VM）
- lab:  192.168.30.0/24（K8s 学习区）
- ai:   192.168.40.0/24（OpenClaw 等高风险区）

## 访问控制原则
- ai -> mgmt：默认禁止
- ai -> lab：仅允许访问 K8s API / 必要服务端口（最小化）
- tools -> lab：允许（用于日常访问监控/面板等）
- 外部访问：优先 VPN/Zero Trust；禁止直接暴露 PVE 管理界面

## DNS / 域名（占位）
- 内网域：*.lab.home / *.tools.home（示例）
- 统一反代入口：proxy.tools.home
