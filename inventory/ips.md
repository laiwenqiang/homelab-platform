# 网络地址规划表（192.168.5.0/24）

## 一、总体网段划分

| 区域 | IP 范围 | 用途说明 |
|------|----------|----------|
| mgmt | 192.168.5.1–19 | PVE 管理区，仅管理终端可达 |
| tools | 192.168.5.20–49 | 日常工具 LXC/VM（反代、监控、DNS、日志等） |
| lab | 192.168.5.50–79 | K8s 学习区 / 实验环境 |
| ai | 192.168.5.80–89 | 高风险区（OpenClaw / 下载类 / 可疑代码） |
| reserve | 192.168.5.90–99 | 未来扩展 |
| clients (DHCP) | 192.168.5.100–199 | 手机 / 笔记本 / 家庭终端 |
| reserve | 192.168.5.200–254 | IoT / Guest / 预留 |

## 四、逻辑分层说明

| 区域 | 风险等级 | 是否可访问 mgmt |
|------|------------|----------------|
| mgmt | 最高权限 | — |
| tools | 低风险基础服务 | ❌ 默认禁止 |
| lab | 中风险实验区 | ❌ 默认禁止 |
| ai | 高风险区 | ❌ 严格禁止 |
| clients | 不可信终端 | ❌ 严格禁止 |
